GitHub

Aspecto Descripción
¿Qué es GitHub? GitHub es una plataforma basada en la nube que permite a desarrolladores y equipos almacenar, compartir y colaborar en proyectos de código. Basada en Git, facilita el control de versiones y la colaboración.
Características
  • Control de versiones con Git
  • Repositorios públicos o privados
  • Colaboración con Pull Requests, Issues, Wikis, Code review
  • Comunidad activa
  • Integraciones con herramientas de desarrollo
  • GitHub Codespaces para desarrollo en la nube
Ventajas
  • Colaboración eficiente
  • Control de versiones
  • Gran comunidad
  • Portafolio para desarrolladores
  • Integración continua
  • Recuperación de errores
Desventajas
  • Curva de aprendizaje
  • Costo para proyectos privados
  • Interfaz menos intuitiva para algunos
Importancia GitHub facilita la colaboración global, mejora la calidad del código, acelera la innovación, y proporciona historia y auditoría del proyecto.
Datos curiosos
  • 100 millones de desarrolladores utilizan GitHub.
  • Más del 90% de las empresas Fortune 100 lo usan.
  • Fue adquirido por Microsoft en 2018.

Hosting

Aspecto Descripción
¿Qué es el Hosting? El hosting permite almacenar contenidos necesarios para que un sitio web y correos sean accesibles en Internet. Es el espacio en un servidor para guardar los archivos del sitio web.
Características
  • Almacenamiento (SSD o HDD)
  • Ancho de banda
  • Tiempo de actividad (Uptime)
  • Conectividad
  • Soporte para scripts y bases de datos
  • Seguridad
  • Soporte técnico
  • Panel de control
Ventajas
  • Rendimiento
  • Espacio y ancho de banda generosos
  • Seguridad avanzada
  • Soporte técnico profesional
  • Dominio propio
  • Sin publicidad no deseada
Desventajas
  • Publicidad no deseada en hosting gratuito
  • Rendimiento limitado
  • Seguridad deficiente
  • Soporte limitado
  • Riesgo de pérdida de datos
Datos curiosos
  • Un "host" es cualquier máquina conectada a la red.
  • El tiempo ideal de carga es menos de tres segundos.
  • El origen del hosting se remonta a la década de 1990.
